# Madmatt06 Tap

## How do I install these formulae?

`brew install madmatt06/tap/<formula>`

Or `brew tap madmatt06/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "madmatt06/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
